# Projeto 2 – Previsão de Renda

Este projeto realiza uma análise exploratória de dados para prever renda com base em variáveis socioeconômicas. A aplicação foi desenvolvida em Python utilizando Streamlit para criar uma interface interativa.

## Arquivos
- projeto 2.py: script principal da aplicação.
- 2025-12-26 10-36-09.mp4: vídeo demonstrativo da aplicação.

## Como executar
1. Instale as dependências:
   pip install streamlit pandas matplotlib seaborn

2. Execute o aplicativo:
   streamlit run "projeto 2.py"

3. Acesse no navegador:
   http://localhost:8501
